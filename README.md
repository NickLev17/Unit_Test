_# Модульное тестирование
 ## Задача
Необходимо реализовать операцию нахождения корней квадратного уравнения и протестировать её.


### Сигнатура тестируемой функции
vector<double> solve(double a, double b, double c, double e)
a, b, c - коэффициенты квадратного уравнения, функция возвращает список корней квадратного уравнения.

### Набор тестов
 NotSquareRoot - тест, который проверяет, что для уравнения x^2+1 = 0 корней нет (возвращается пустой массив).
 TwoSquareRoot-  тест, который проверяет, что для уравнения x^2-1 = 0 есть два корня кратности 1 (x1=1, x2=-1)
 OneSquareRoot - тест, который проверяет, что для уравнения x^2+2x+1 = 0 есть один корень кратности 2 (x1= x2 = -1)
 ArgumentAException- тест, который проверяет, что коэффициент a не может быть равен 0. В этом случае solve выбрасывает исключение. Учесть, что a имеет тип double и сравнивать с 0 через == нельзя.
 AllArgumentException - тест, который проверяет корректность всех аргументов функции.
